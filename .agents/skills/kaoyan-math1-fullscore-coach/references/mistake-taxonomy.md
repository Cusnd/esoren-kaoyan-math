# 错因分类表

处理错题或用户疑问时，将错误归入一个或多个类别。

## 高等数学

1. 定义不清：极限定义，连续/可导/可微关系，反函数，复合函数，级数收敛概念。
2. 定理条件漏检：洛必达条件，泰勒余项，中值定理条件，积分换元条件，反常积分收敛条件。
3. 公式记错：等价无穷小，求导公式，积分公式，级数展开。
4. 计算错误：代数化简，符号错误，上下限错误，Jacobian 错误。
5. 变量混乱：参数与最终变量混淆，换元后未改上下限，多元变量含义混乱。
6. 题型识别失败：该用中值定理却硬算，该换序却直接积分，该整体换元却硬解反函数。

## 线性代数

1. 矩阵维度错误。
2. 行变换与列变换混用。
3. 秩与解的关系不清。
4. 特征值、相似、合同混淆。
5. 基础解系个数判断错误。
6. 正定判别条件漏检。

## 概率论与数理统计

1. 事件关系判断错误。
2. 独立性误用。
3. 条件概率方向错误。
4. 密度积分区域错误。
5. 分布函数边界错误。
6. 期望方差公式误用。
7. 统计量与参数混淆。
8. 似然函数构造错误。

## 做题策略错误

1. 第一入口选错。
2. 复杂题没有拆步骤。
3. 过早代入数字。
4. 没有检查特殊情况。
5. 没有验证答案范围。
6. 考场书写不规范导致扣分。
